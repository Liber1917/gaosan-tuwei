# 只有高考：高三决策原则

## Rule（规则）
高三一切行为的唯一标准是：这件事对高考有没有帮助？所有其他考量——排名、评价、娱乐权衡、模拟考情绪——都是对高考注意力的稀释。简化为"只有高考"，就能自动消除大量决策疲劳和情绪消耗。

## Instruction（操作指南）

### "只有高考"的三层含义
1. **除了高考，其他一切都不重要**
2. **除了能让高考变得更好的，其他一切都不重要**
3. **阻碍你完成终极目标的一切都不重要**

### 七个自检问题（对号入座）
以下任何一条符合，说明你的"眼中不只有高考"：

| 行为 | 说明你在意的其实是 |
|------|------------------|
| 为模拟考排名紧张失落 | 模拟考排名 |
| 拿自己和同学比较 | 相对位置 |
| 还在问"现在要不要刷真题" | 找理由偷懒 |
| 每次考试"水一水，下次再努力" | 暂时的舒适 |
| 还在权衡要不要戒手机/游戏 | 娱乐 |
| 因为天赋差距而自我怀疑 | 不可改变的因素 |
| 还在整天刷知乎找方法 | 完美主义的借口 |

### "只有高考"的行为转化
- **考试策略**：每次模拟考 = 高考演练，只关注"发现了什么漏洞"，不关注排名
- **时间策略**：每天决策时问"这件事让我距高考更近还是更远？"
- **娱乐策略**：手机/游戏不是"应不应该戒"的问题，而是"本来就不应该在视野里"
- **比较策略**：和同学比较结果没有意义，因为"比较的不是高考成绩"

### 情绪触发时的应急程序
1. 发现情绪波动
2. 立刻停下，问："是什么占据了不该占据的位置？"
3. 评估：除了亲人健康，真的有比高考更重要的吗？
4. 确认后：切回学习状态，把刚才的情绪时间当作噪音过滤掉

## Action（即时行动）
在今天结束前，把手机放到另一个房间，不带进书房/学习区域。
这不是"应该做的事"，这是"只有高考"的自然结果——手机本来就不属于那个空间。
