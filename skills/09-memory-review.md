# 防遗忘复习系统

## Rule（规则）
遗忘是生理规律，不是努力不够的证据。遗忘曲线告诉我们：学习后的遗忘在第1天内速度最快，之后逐渐放缓。在关键时间节点及时复习，可用极少时间维持极高的记忆保留率。

## Instruction（操作指南）

### 遗忘曲线关键节点
| 复习时机 | 距学习时间 | 记忆保留率 |
|---------|----------|-----------|
| 第1次复习 | 学后1天 | 从约33%→提升至接近100% |
| 第2次复习 | 学后3天 | 再次强化，电线开始加粗 |
| 第3次复习 | 学后7天 | 进入稳定区间 |
| 第4次复习 | 学后14天 | 向长期记忆转化 |
| 第5次复习 | 学后30天 | 基本固化为长期记忆 |

**核心原则**：及时复习 > 事后补救。复习投入的时间少，但效果大于首次学习的3倍。

### 高三可行的复习节奏

**每日复习（5-10分钟）**
- 当天学完新内容，睡前快速过一遍笔记（不需要重新理解，只需激活记忆）
- 重点关注：今天课上没听懂的点、老师特别强调的重点

**每周复习（30-60分钟）**
- 周末把本周所有学科的新知识串联复习一遍
- 使用"知识树"或"思维导图"做跨章节联系
- 检查本周做错的题，确认是否已经理解

**每月复习（2-3小时）**
- 月底对本月所有新内容进行系统回顾
- 重做一遍本月错题本中的题，确认是否真正掌握
- 把本月零散的笔记整理成结构化的总结

### 记忆类科目专项策略

**文综/历史/地理/政治**
- 制作时间轴/思维导图，比背诵散点知识效率高5倍
- 每个知识点配"关键词"，复习时只看关键词触发完整记忆
- 高频复习容易混淆的近似概念

**英语单词**
- 使用间隔重复工具（如Anki思路手动执行：按上述节点复习生词）
- 每次遇到生词，当天晚上放入"待复习池"，按1/3/7/14/30天节点复习
- 不认识的单词分级标记（3次不会=重点记忆）

**数学公式/定理**
- 不是背公式，是"背推导过程"（理解来源→自然记住公式）
- 高频使用的公式靠"练"形成肌肉记忆，不靠强记

### 如何判断"已经记住了"
**达到以下标准才算真正掌握**：
- 闭眼能完整复述该知识点
- 换一种题目形式依然能识别并应用
- 隔3天后再看，无需重新理解

## Action（即时行动）
为今天学习的一个新知识点，设置3个复习提醒：明天、3天后、7天后。
用手机日历或便利贴完成。这个动作花时间不超过2分钟，但会让今天的学习成果保留率提高3倍。
