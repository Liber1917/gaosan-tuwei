# 考场作战策略

## Rule（规则）
考场上的策略失误造成的丢分，和知识不会造成的丢分一样真实。时间管理不当、难题卡壳不肯放手、检查环节缺失——这些都是可以通过演练消除的可控因素。

## Instruction（操作指南）

### 考前准备清单

**考前一周**：
- [ ] 确认考试时间、地点、准考证
- [ ] 演练：用真题在完整时间内模拟考试（含写名字、检查环节）
- [ ] 检查文具：2B铅笔（涂答题卡）、中性笔（0.5黑色）、直尺、圆规

**考前三天**：
- [ ] 轻量复习（只看"考前必看清单"，不学新东西）
- [ ] 保证睡眠（即使失眠也不要焦虑，平躺闭眼就是休息）
- [ ] 不对答案，不和同学讨论复习内容

**考前一天**：
- [ ] 下午3点后停止学习，整理文具，放松
- [ ] 不刷手机，不看新内容
- [ ] 早睡（即使睡不着也要早躺下）

### 考场时间分配原则

**通用策略**：先易后难，不在一道题上耗费超过预设时间

| 题型 | 时间分配比例 | 超时判断标准 |
|------|------------|------------|
| 选择题/填空题 | 约30-40% | 单题超过1.5倍正常时间 → 标记跳过 |
| 大题前几问 | 约40-50% | 超过该题总分×1.5分钟 → 先做下一问 |
| 大题后几问/压轴 | 约15-20% | 酌情处理，保证前面题目的完整性 |
| 检查 | 保留约10% | 不可省略 |

**检查优先级**：
1. 答题卡有没有漏涂
2. 是否有空白未填
3. 大题步骤是否完整（有没有写"答：..."）
4. 最后才是验算

### 遇到难题的应急处理

**三步处理法**：
1. **30秒扫描**：读题，判断是否有思路
2. **有思路**：继续做；**没思路**：果断标记，跳过
3. 全部会做的题答完后，回来集中处理标记题

**处理压轴题（如数学最后两题）**：
- 后面题的前几问通常较简单，先做前几问拿保底分
- 最后一问（极难）不会做时，写出已知条件和推导方向，可能得到步骤分
- **不要**：花20分钟死磕一道压轴题，导致前面的题来不及检查

### 考试中的心态管理

**发现题目很难时**：
"题目难对所有人都难，影响的是相对排名，不只是我。专注当下这道题能得多少分。"

**遇到完全不会的题时**：
"这道题我不会，写上相关公式和已知推论，争取步骤分。这不代表考差了，继续下一题。"

**翻页发现时间不够时**：
先填写所有你有答案的题（确保能得的分都得），再回来处理难题。

### 考后处理原则

**绝对不要做的事**：
- 走出考场后立刻对答案（影响下一科状态）
- 讨论"那道题你怎么做的"
- 计算估分（引发焦虑，浪费切换时间）

**应该做的事**：
- 走出考场，深呼吸，告诉自己"这科结束了"
- 补充能量（零食/水）
- 心理切换到下一科（如果同天还有考试）

## Action（即时行动）
下次模拟考结束后，做一件事：
不对答案，直接写下"这次考试中我的时间安排哪里出了问题"。
针对这个问题，在下次考试中做出调整。
这比对答案计分更有价值。
