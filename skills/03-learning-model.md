# 灯泡-电线学习模型

## Rule（规则）
知识点对应大脑神经元（灯泡），知识联系和解题套路对应突触连接（电线）。学习和复习的目标，是让灯泡又多又亮、电线又密又粗且不消失。

## Instruction（操作指南）

### 五条神经科学学习规律

| 规律 | 含义 | 实践操作 |
|------|------|----------|
| ① 重复→灯泡变亮 | 对一个知识点重复接触、深入理解，记忆越牢 | 同一知识点多次复习、用不同题型练习 |
| ② 同时激活→自动连线 | 两个知识点一起使用，自动建立联系 | 做综合题、跨章节专题训练 |
| ③ 反复激活→电线加粗 | 经常一起用的知识点，联系越来越稳固 | 专项刷题、错题反复练习 |
| ④ 长期不用→电线消失 | 长时间不复习，知识联系会退化 | 按艾宾浩斯曲线定期复习 |
| ⑤ 电线够粗→长期记忆 | 复习到一定程度后形成长期记忆，不再遗忘 | 坚持到"闭眼会做"再停止重复 |

### 解题过程的神经科学解释
1. 读题审题 → 激活与题干相关的起点灯泡
2. 起点灯泡通过电线带亮整个相关知识网络
3. 答题所需的所有灯泡都亮起 → 顺利解题
4. 某个灯泡不亮（知识漏洞）或电线断开（思路断层）→ 卡壳

**做题卡壳的两个原因**：
- A. 缺灯泡：知识点没有学过或忘记
- B. 缺电线：知识点知道但不知道怎么联系/应用

两种原因的补救方式完全不同，要先判断是哪种再补救。

### 学霸的本质
脑中灯泡足够多且亮（知识覆盖广且深）+ 电线密集粗壮（综合应用能力强）。
天赋学霸：灯泡和电线一点即亮一点即稳。
普通学生：需要多次重复，但最终能达到同样状态。**差距只在速度，不在终点。**

### 如何用这个模型指导复习计划
- 每次做错题，先判断：缺灯泡（补知识点）还是缺电线（补题型套路）？
- 新学内容 = 点亮新灯泡（理解为主）
- 复习 = 反复激活现有灯泡+电线（加粗加固）
- 综合练习 = 检查各区域电线连通性（查漏）

## Action（即时行动）
拿出最近一次做错的题，判断：
- 是"没见过这个知识点"（缺灯泡）→ 补课本
- 还是"知识点会，但不知道怎么用"（缺电线）→ 补这类题的套路

然后针对性地填补，不要两种情况都用"多做题"解决。
