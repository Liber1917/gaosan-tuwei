# 内容鄙视链：高效筛选学习资源

## Rule（规则）
不是所有的学习资源都值得投入时间。按价值排序：思想 > 方法论 > 行动建议 > 经历分享 > 抒情故事。高三时间有限，应优先摄取最高密度的有价值内容，拒绝被鸡汤感动却不改变的陷阱。

## Instruction（操作指南）

### 五级内容价值体系

**第1级：思想（最高价值）**
- 特征：开拓思维，改变你看问题的方式，能长期持续影响行动
- 举例："高考是个概率游戏，我每天要做的是提高考好概率" → 改变了你对每次考试的态度
- 吸收方法：精读，深度思考，用自己的语言复述，找到对自己的具体含义

**第2级：方法论（很高价值）**
- 特征：针对某类问题的系统解决方案，既说"是什么"也说"为什么"，还指出注意事项
- 举例："30天方法训练术"三阶段框架
- 吸收方法：完整理解框架，然后原样试用，再根据实际情况改良

**第3级：行动建议（较高价值）**
- 特征：分条目告诉你"该怎么做"，但不解释为什么
- 问题：没有原因，记不住，用不上，"明明知道但做不到"
- 吸收方法：找到每条建议背后的道理（自己推导），建立真正的理解

**第4级：经历分享（较低价值）**
- 特征：大篇幅描述个人经历，信息密度低
- 问题：需要从故事里抽丝剥茧提炼干货，效率极低
- 使用方法：只读"总结/结论"部分，跳过叙事段落

**第5级：抒情故事/鸡汤（极低价值）**
- 特征：感动你、激励你，但不告诉你具体怎么做
- 问题：鸡血效果只持续1天，第二天原地复原
- 使用方法：高三不建议阅读，把时间用于第1-2级内容

### 快速判断一篇文章的价值（30秒测试）
1. 读完这篇文章，我会改变什么行为吗？（能改变 → 第1-2级）
2. 文章告诉我具体怎么做了吗？（有具体操作 → 第2-3级；只有"要努力" → 第4-5级）
3. 这些内容是可以验证的吗？（可验证 → 高价值；纯故事 → 低价值）

### 识别"伪干货"答主
套路：编造逆袭故事（300分→600分）→ 抛出正确废话 → 搬运别人的方法论改写表达 → 灌鸡汤收粉
判断标准：内容是否具体可操作？是否给出过为什么？故事可否独立验证？

**真正有价值的作者**：观点体系化、有自己独特视角、干货密度高、逻辑推导清晰。

## Action（即时行动）
打开你手机上最近收藏的一篇学习方法文章，用30秒测试评估它的价值等级。
如果是第4-5级，果断关闭，把这段时间用来做一道题。
