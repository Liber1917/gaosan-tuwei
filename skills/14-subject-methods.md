# 各学科核心方法速查

## Rule（规则）
不同学科有不同的核心考查维度，用错方法事倍功半。了解每门学科的本质逻辑，才能在正确方向上积累晶体智力，而非在边缘问题上浪费时间。

## Instruction（操作指南）

---

### 数学

**本质**：考查逻辑推理能力 + 题型套路熟练度（晶体智力密集区）

**核心策略**：
1. **题型分类管理**：把高考数学的所有题型整理成类目（函数/数列/三角/向量/解析几何/概率/空间几何），每类题型有固定解题框架
2. **基础计算必须快准**：每天10分钟计算训练（导数/积分基础运算），速度决定时间余量
3. **从真题中提炼套路**：最近10年高考数学真题全部做完，每道题分析"考的是哪个知识点，用什么套路"
4. **高难题策略**：最后两道题的第一问必须拿到，最后一问允许放弃（时间不够时）

**易犯错误**：
- 过于依赖临场推导，不积累套路（流体智力陷阱）
- 做了大量偏题怪题，忽视高频考点的熟练度
- 解析几何不设坐标系就开始算（习惯问题）

---

### 语文

**本质**：考查语言理解能力 + 表达能力 + 文化积累

**核心策略**：
1. **阅读理解答题框架**：
   - 现代文：审题关键词→定位原文→按分点答题（不超过字数限制）
   - 古文：实词虚词积累→断句逻辑→结合上下文猜意思
2. **作文素材积累**：建立素材库（社会议题/历史人物/科学案例），每个素材配3个不同角度的使用方法
3. **高考真题研究**：重点分析阅卷标准——答题点在哪里，扣分原因是什么
4. **诗词/文言文**：不靠死记硬背，靠理解背景+逻辑推导含义

**高频失分区**：
- 古诗文默写空白（低成本高回报的背诵区）
- 现代文阅读"答了但没对上点"
- 作文开头不凝练，结尾不回扣主题

---

### 英语

**本质**：语言工具，考查词汇量 + 阅读速度 + 输出能力

**核心策略**：
1. **词汇**：每天固定背单词，优先背考纲高频词，不追求生僻词
2. **阅读**：训练"找主旨句+关键词"的快速定位，不逐字翻译
3. **写作**：熟记并活用10-15个高分句型，背3-5篇范文并内化（能用自己的话复现）
4. **听力**（如有）：每天固定听英语材料15分钟，培养语感而非靠考前突击

**提分最快的区域**：
- 完形填空：做完多读一遍全文，逻辑通顺才对
- 语法填空：先读全段把握语境，再处理每个空格
- 书面表达：固定格式+高分句型，字数充足，不犯低级语法错误

---

### 理综（物理/化学/生物）

**本质**：考查概念理解 + 实验能力 + 综合应用

**物理核心**：
- 受力分析是一切的基础，先把力画出来再列方程
- 图像题：先认出"这是什么图""斜率/面积代表什么"
- 不会解题时，先写已知量和未知量，往往推导路径自然浮现

**化学核心**：
- 实验题有固定套路（操作→现象→结论→误差分析）
- 有机化学推断题：从官能团逆推，确定框架后填细节
- 电化学：背核心反应方程式，在此基础上推导变体

**生物核心**：
- 教材中的每个关键词都是考点，不要跳过"不重要的"定义
- 遗传题：画遗传图（不要直接在脑子里算），步骤分可能更多
- 实验题：实验设计遵循"单一变量"原则，结论只对变量负责

---

### 文综（历史/地理/政治）

**本质**：考查记忆广度 + 应用分析能力 + 主观题答题框架

**历史核心**：
- 时间轴是核心工具，每个时期建立"政治/经济/文化"三维框架
- 材料题：找关键词→定位史实→论据+论点结合材料作答
- 高频考点：中国近代史、世界大战前后的政治格局变化

**地理核心**：
- 区域地理：地图识记是基础，能在脑中"还原"地图再答题
- 自然地理：有原理可循，理解地理过程（如水循环、洋流成因）
- 答题套路：自然条件/人文条件分开答，多角度给分

**政治核心**：
- 主观题使用"政治术语"而非口语表达，答题模板非常重要
- 每个"原理"对应固定"方法论"，背"原理-方法论"对比表
- 时事结合点：热点与课本知识的结合角度提前整理

## Action（即时行动）
选择你现在最弱的一门学科，对照上面的"易犯错误"或"高频失分区"，找出你现在最主要的问题是哪一个，今天针对这一个点做专项练习。
