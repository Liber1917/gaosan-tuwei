# 错题本深度运营指南

## Rule（规则）
错题是知识网络的断点，修复每一个断点等于加粗一条关键电线。错题本的价值不在于"收集"，而在于"内化"。收集是起点，二刷是关键，能在不看答案的情况下独立做对才是终点。

## Instruction（操作指南）

### 错题收录标准（哪些题该进错题本）

**✅ 该收录**：
- 看完题目完全没有思路（知识漏洞）
- 有思路但过程中走错（套路漏洞）
- 同类型题反复出错（系统性问题）

**❌ 不该收录**：
- 计算粗心出错（写"粗心"标签另行管理）
- 会做但笔误（这类题在别的地方管理）
- 看完解析立刻明白的"会了"的题（没必要）

**粗心题的单独管理**：建立"粗心记录表"，记录粗心类型（数字抄错/漏看条件/步骤跳步），每月统计高频粗心类型，针对性培养检查习惯。

### 错题记录格式

每道错题记录以下内容：
```
【题目来源】：XX模拟/真题/XX章练习
【题目类型】：（例：数学-二次函数最值问题）
【错误原因】：（选择：知识漏洞 / 套路漏洞 / 思维习惯问题）
【具体问题】：（例：不知道"对称轴在区间内/外"要分类讨论）
【正确思路】：（用自己的话写，不是抄答案）
【触发条件】：（例：看到"二次函数+有界区间+求最值"想到分类讨论）
【二刷状态】：□第1次 □第2次 □第3次（独立完成）→ 划掉
```

### 二刷执行策略

**二刷频率**：
- 每2周对当前错题本进行一次完整二刷
- 二刷时完全不看原题答案，用新草稿纸独立完成
- 做对 → 打勾（√），继续下一题
- 还做不对 → 重新分析原因，标红，下次优先复习

**毕业条件**：连续2次独立做对，无需回头看正确思路，该题可从"重点复习"列表中移除。

**关键原则**：错题本是用来"做"的，不是用来"看"的。看了没用，做了才有效。

### 错题本的物理形式建议

**纸质错题本**：
- 按学科分开，每科一本
- 横线本优于白纸（保持整洁，减少分心）
- 写完一题留3行空白，方便后续补注

**电子错题本**（可选补充）：
- 拍照归档（题目照片+手写分析）
- 优点：便于搜索、不占物理空间
- 缺点：做题时不如纸质方便

**不推荐**：直接在书上圈画（无法二刷）；把所有错题打印贴纸（工程量大且无法自己写思路）

### 高考前的错题本使用策略

**考前1个月**：全面三刷所有错题本，重点攻克"标红"题
**考前2周**：只做从未独立完成过的高难度错题
**考前3天**：只浏览各题的"触发条件"，激活记忆，不做题

## Action（即时行动）
翻出你最近一次考试的试卷，找出3道"不会做"的错题。
用上面的格式，每题花5分钟写完记录。
3题记录完毕后，设2周后的提醒，到时二刷。
