# 临考冲刺30天手册

## Rule（规则）
高考前30天，策略比努力更重要。这30天不适合学新东西，而适合回归基础、稳定发挥、消除已知问题。心态稳定是这30天最高价值的投入，而非死磕难题。

## Instruction（操作指南）

### 30天整体策略

**总原则**：减少新输入，最大化已有知识的调取稳定性

| 阶段 | 时间 | 核心任务 |
|------|------|---------|
| 第1-15天 | 高考前30-15天 | 全真模拟 + 专项补漏 |
| 第16-25天 | 高考前14-5天 | 回顾错题本 + 整理考前清单 |
| 第26-27天 | 高考前4-3天 | 轻量复习，睡眠优先 |
| 第28-29天 | 高考前2-1天 | 只看考前清单，整理文具 |
| 第30天 | 高考当天 | 执行考场策略 |

### 第1-15天：全真模拟阶段

**每2-3天进行一次全真模拟考试**：
- 严格按高考时间（不能超时）
- 在安静环境下完成（模拟考场压力）
- 考后分析错误类型，不对答案计算分数

**每次模拟后必做的事**：
1. 列出错误清单（按学科分）
2. 判断每个错误：知识漏洞/套路问题/心态失误/粗心
3. 知识漏洞 → 立刻补，当天解决
4. 粗心问题 → 记录类型，训练检查习惯

### 第16-25天：系统回顾阶段

**每天回顾错题本**：
- 做3轮快速回刷（不做题，只看"触发条件"和"解题框架"）
- 将高频错误的类型记在"考前重点清单"上

**整理六科"考前必看清单"**：
每科一张A4纸，内容包括：
- 核心公式/定理（不能错的）
- 高频题型套路（最容易混淆的）
- 本人历史高频失误点（个性化）

这张清单是考前3天的唯一复习材料。

### 第26-27天：减速阶段

- 每天学习时间控制在4-5小时（比平时少）
- 内容：只看考前必看清单，不做新题
- 睡眠：提早入睡，保证8小时
- 饮食：规律，不吃可能引发不适的食物
- 心态调节：允许自己有轻微焦虑，这是正常的应激反应

### 第28-29天：考前准备

**第29天（考前2天）**：
- 早上轻量活动
- 下午：准备考试文具、查看考场地址、规划出行路线
- 不学习新内容
- 晚上正常入睡

**第28天（考前1天）**：
- 上午：过一遍考前必看清单（1小时）
- 下午：放松（散步、听音乐、不看书）
- 晚上：10-11点前入睡，不强迫自己快速入睡

### 临考焦虑处理

**焦虑是正常的生理反应**：
- 适度焦虑（肾上腺素分泌）会让大脑更清醒
- 不要试图"完全消除"焦虑，而是接受它、利用它

**过度焦虑（影响睡眠/注意力）的处理方法**：
1. **重框架**：高考只是一次考试，它影响路径但不决定人生
2. **聚焦可控**：把注意力放在"今天我能做什么"而非"如果考差了怎么办"
3. **身体调节**：深呼吸（4秒吸-4秒屏-4秒呼）；有氧运动20分钟

**失眠处理**：
- 平躺闭眼 = 生理休息，不需要真的睡着
- 不要反复看时间（加剧焦虑）
- 大脑即使没有深睡眠也能在高考当天正常发挥（经过反复模拟训练后）

## Action（即时行动）
**如果距高考 > 30天**：立刻制定自己的三轮复习计划，确定每一轮的截止时间。
**如果距高考 ≤ 30天**：今天制作第一张"考前必看清单"，选择你最薄弱的一门学科，把核心公式和高频错误写在一张A4纸上。
